📖 Instruções Completas

  • 1. Preparar o Celular
    Vá em Configurações → Segurança → Ative "Instalar de fontes desconhecidas"
  • 2. Baixar o APK
    Clique no botão "Baixar APK" e aguarde completar
  • 3. Instalar
    Abra o Gerenciador de Arquivos → Downloads → diamanteblack.apk → Instalar
  • 4. Primeiro Uso
    Abra o app e vá em Configurações (⚙️) para baixar os modelos de IA
  • 5. Modelos Disponíveis
    • 🧠 Qwen2 (200 MB) - Lógica
    • 💻 TinyLlama (500 MB) - Código
    • ✨ SmolLM (100 MB) - Síntese
    • 🧬 Phi-1.5 (800 MB) - Raciocínio
  • 6. Requisitos Mínimos
    • Android 7.0+
    • 2GB RAM mínimo
    • 2GB espaço livre
    • Internet para baixar modelos
